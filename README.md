# Nana Kotaa School Complex Website

A simple, professional, and mobile-friendly website for Nana Kotaa School Complex.

## Technologies Used
- HTML5
- SCSS (compiled to CSS)
- JavaScript

## Project Structure
```
/
├── index.html          # Home page
├── about.html          # About page
├── staff.html          # Staff page
├── academics.html      # Academics page
├── gallery.html        # Gallery page
├── contact.html        # Contact page
├── scss/
│   └── main.scss       # Main SCSS file
├── css/
│   └── main.css        # Compiled CSS
├── js/
│   └── main.js         # JavaScript for animations
└── images/             # Place your images here
```

## Setup Instructions

1. Clone or download the project files.
2. Place your school images in the `images/` folder:
   - `school-hero.jpg` for the hero section
   - `staff1.jpg`, `staff2.jpg`, etc. for staff photos
   - `gallery1.jpg` to `gallery6.jpg` for gallery images
3. Update the content in HTML files with your actual school information.
4. If you need to modify styles, edit `scss/main.scss` and recompile using:
   ```
   npx sass scss/main.scss css/main.css
   ```
